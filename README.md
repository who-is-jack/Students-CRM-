# 留学规划 · 学生档案 CRM 系统

> 专为留学规划师设计的学生背景管理工具，支持六大背景模块记录、AI 简历生成、待办提醒、Excel 批量导入导出等功能。

---

## 功能概览

| 模块 | 功能 |
|------|------|
| 📋 学生档案 | 建档、编辑、按申请季筛选 |
| 📚 六大背景模块 | 教育背景、语言成绩、竞赛、实习、科研、交流交换 |
| ✦ AI 简历生成 | 中英文简历，内置 STAR 法则润色 Prompt |
| ☑ 待办提醒 | 每个学生独立待办 + 全局待办汇总视图 |
| 📎 附件上传 | 支持 PDF、Word、图片等文件本地存储 |
| ↓ 数据导出 | 一键导出 CSV，可用 Excel 打开 |
| ⬆ 批量导入 | 下载 CSV 模板，填写后批量上传学生信息 |

---

## 技术栈

- 纯 HTML + CSS + JavaScript 单文件应用
- 数据存储：`localStorage`（本地浏览器，无需服务器）
- AI 功能：Anthropic Claude API（`claude-sonnet-4-20250514`）
- 部署：GitHub + Vercel（免费）

## 使用方法

### 1. 设置 API Key

AI 简历生成功能需要 Anthropic API Key：

1. 打开 [console.anthropic.com](https://console.anthropic.com) 注册登录
2. 左侧 **API Keys** → **Create Key** → 复制 `sk-ant-...` 开头的密钥
3. 粘贴到网站**左下角侧边栏**的 API Key 输入框
4. 显示「✓ 已设置」即可，Key 保存在本地浏览器中

### 2. 新增学生

- 点击右上角「＋ 新增学生」，填写基本信息和初始背景
- 或点击「⬇ 下载 CSV 模板」，批量填写后「⬆ 上传 CSV」批量导入

### 3. 编辑学生档案

点击学生列表中任意一行，侧边详情面板包含：

- **背景总览**：教育背景、语言成绩、联系&申请季（均可编辑）
- **活动经历**：竞赛/实习/科研/交流，每条经历卡片式录入（标题 + 时间 + 详细描述）
- **附件文件**：上传简历、成绩单等文件
- **待办提醒**：为该学生添加待办事项，支持截止日期
- **沟通记录**：按时间顺序记录每次沟通内容
- **简历润色**：基于六大背景模块一键生成中英文简历

### 4. 申请季筛选

学生列表顶部支持按申请季快速筛选：**27Fall / 28Fall / 29Fall / 30Fall**

## AI 简历生成规则

内置润色 Prompt，生成逻辑如下：

- 用 **STAR 法则**隐性嵌入（情境 / 任务 / 行动 / 结果）
- 每条经历总结成一句，突出「做了什么 → 怎么做 → 带来什么结果」
- 增加**量化指标**和具体的软件工具、技术栈
- 以本科生能力为边界，不虚构事实，用「约 X 次」「约 X 万条」等方式补齐量化
- 支持**英文简历**和**中文简历**两种语言
- 支持学术型 / 专业型 / 国际化精炼 / 突出实习四种风格

## 数据说明

- 所有学生数据存储在**本地浏览器 `localStorage`** 中，不上传任何服务器
- API Key 同样仅存于本地，不外传
- 清除浏览器缓存会导致数据丢失，建议定期用「导出 CSV」功能备份

## 部署方式（GitHub + Vercel）

### 首次部署

1. 登录 [github.com](https://github.com)，新建仓库（建议命名 `studyabroad-crm`，选 Public）
2. 进入仓库 → **Add file** → **Upload files**，上传 `index.html`（将文件重命名为 `index.html`）
3. 登录 [vercel.com](https://vercel.com)，用 GitHub 账号登录
4. **Add New Project** → 选择该仓库 → **Deploy**
5. 约 30 秒后部署完成，获得访问网址（如 `studyabroad-crm.vercel.app`）

### 后续更新

每次修改代码后：

1. 进入 GitHub 仓库，点击 `index.html`
2. 点右上角**铅笔图标（Edit）**
3. 将新代码完整粘贴替换
4. 点 **Commit changes**
5. Vercel 自动检测并在约 1 分钟内完成更新 ✅

## v2.6 — 活动经历卡片化 + 已完成申请功能
*本次更新*

### ✨ 新功能

**活动经历升级为卡片式录入**
- 竞赛、实习、科研、交流交换四个模块全面升级
- 每条经历独立一张卡片，包含三个字段：**标题**（机构/赛事名）、**时间**、**详细描述**
- 详细描述支持多行自由填写，不再受单行限制
- 每张卡片可单独**编辑**或**删除**，操作更灵活
- 自动兼容旧数据迁移，历史记录不会丢失

**「完成申请」功能**
- 学生详情面板顶部新增「✓ 完成申请」按钮
- 点击后弹出自定义确认框，确认后学生移入「已完成」分类
- 已完成学生从原申请季列表消失，只出现在「✓ 已完成」筛选分类中
- 支持「↩ 撤销完成」，可随时恢复至原申请季
- 学生列表中已完成学生显示绿色「✓ 已完成」标签，并保留原申请季小字

**仪表盘更新**
- 新增「已完成申请」统计卡，显示累计完成人数
- 27Fall / 28Fall 统计数字自动排除已完成学生

### 🐛 Bug 修复

- 修复「完成申请」按钮点击无反应的问题（原因：浏览器在部分环境下静默屏蔽原生 `confirm()` 弹窗）
  - 解决方案：替换为自定义确认弹窗，不依赖浏览器原生对话框
- 修复仪表盘报错 `Cannot set properties of null`（原因：`st-todo` 元素已移除但 JS 仍在读取）

## v2.5 — AI 简历润色 + API Key 设置
*上一版本*

### ✨ 新功能

**内置 STAR 法则简历润色 Prompt**
- 每条经历用 STAR 法则隐性组织（情境 / 任务 / 行动 / 结果）
- 突出「做了什么 → 怎么做 → 带来什么结果」
- 自动补充量化指标（约 X 次 / 约 X 万条）和工具技术栈
- 以本科生能力为边界，合理推断，不虚构事实
- 中英文简历均适用

**侧边栏 API Key 设置入口**
- 左下角新增 Anthropic API Key 输入框
- 输入后自动验证格式，显示「✓ 已设置」
- Key 存储在本地浏览器，刷新不丢失，仅自己可见
- 解决了之前简历生成报「请求失败」的问题

### 🐛 Bug 修复
- 修复 API 调用缺少 `x-api-key` 请求头导致 401 错误
- 修复多处 JS 语法错误（模板字符串中的字面量换行符问题）

## v2.2 — 批量导入 + 附件上传 + 待办提醒 + 中文简历

### ✨ 新功能

**Excel / CSV 批量导入**
- 下载 CSV 模板，按表头填写后一键上传
- 自动跳过重复学生（姓名 + 学校相同）
- 内置 CSV 解析器，无需外部依赖

**数据导出**
- 一键导出 CSV，包含所有字段
- UTF-8 BOM 编码，Excel 打开不乱码

**附件文件上传**
- 每个学生档案支持上传 PDF、Word、图片等文件
- 文件存储于本地浏览器，支持下载和删除

**待办提醒**
- 每个学生独立待办列表，支持截止日期
- 左侧导航新增全局待办视图，汇总所有学生未完成事项
- 逾期显示红色，即将到期显示橙色

**中文简历**
- 简历生成新增中文 / 英文语言切换
- 中英文使用独立的 Prompt 逻辑
